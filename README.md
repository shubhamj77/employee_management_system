# Employee Management System

A full-featured web-based dashboard for managing employee records—add, view, edit, delete, and search employees. Built as a final-year BTech AIML project to demonstrate front-end skills in HTML, CSS, and JavaScript for data handling.

[![Demo](https://img.shields.io/badge/Live_Demo-View-brightgreen)](https://yourusername.github.io/employee-management-system/) <!-- Replace with your GitHub Pages URL after deploying -->

## 🚀 Features
- **CRUD Operations**: Create, read, update, delete employee profiles (name, role, salary, department, etc.).
- **Search & Filter**: Real-time search by name, role, or department.
- **Responsive Design**: Works on desktop, tablet, and mobile.
- **Data Persistence**: Local storage for saving changes (no backend needed).
- Pure vanilla HTML/CSS/JS—no frameworks for lightweight performance.

## 🛠 Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: Browser LocalStorage
- **Tools**: VS Code, Git/GitHub
